# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in the `dbt-checks` community a harassment-free experience for everyone, regardless of:

- age
- body size
- disability
- ethnicity
- gender identity and expression
- level of experience
- nationality
- personal appearance
- race
- religion
- sexual identity and orientation

We are committed to creating an open, welcoming, inclusive, and respectful environment.

---

# Our Standards

Examples of behavior that contributes to a positive environment include:

- being respectful and constructive
- accepting feedback gracefully
- focusing on collaboration
- showing empathy toward other community members
- helping create a welcoming environment
- assuming good intent when possible

Examples of unacceptable behavior include:

- harassment or discrimination
- trolling or insulting comments
- personal or political attacks
- public or private harassment
- publishing private information without permission
- intentionally disruptive behavior
- hostile or exclusionary conduct

---

# Scope

This Code of Conduct applies to:

- GitHub issues
- pull requests
- discussions
- project documentation
- community interactions related to the project

---

# Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing community standards.

Maintainers may remove, edit, or reject comments, commits, code, issues, or contributions that violate this Code of Conduct.

---

# Reporting Issues

If you experience or witness unacceptable behavior, please report it privately to the maintainers.

Security-related concerns should be reported through the process described in `SECURITY.md`.

---

# Enforcement

Project maintainers may take any action they deem appropriate in response to violations of this Code of Conduct, including:

- warnings
- temporary bans
- permanent bans from participation
